## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the Application using vite
   ```sh
   npm run dev
   ```
