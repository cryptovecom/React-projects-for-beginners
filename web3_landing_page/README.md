## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Get a free API Key at [https://example.com](https://example.com)
2. Clone the repo
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the Application using vite
    ```sh
    npm run dev
    ```
